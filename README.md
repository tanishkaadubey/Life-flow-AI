# LifeFlow AI

LifeFlow AI is an AI-powered productivity dashboard designed to help users manage tasks, track productivity, and improve workflow efficiency through intelligent insights and automation.

## Features

* AI-powered productivity dashboard
* Task management system
* Interactive analytics & charts
* Dynamic user profile system
* Smart AI insights & recommendations
* Modern glassmorphism UI
* Responsive design
* Real-time task statistics
* Settings management panel
* Dark futuristic interface

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Framer Motion
* Recharts
* Lucide React Icons

### Backend

* Supabase

## Dashboard Features

* Productivity tracking
* AI accuracy monitoring
* Pending & completed task analytics
* Weekly productivity flow chart
* Recent task preview table

##  Settings Features

* Dark mode toggle
* Notification settings
* AI preferences
* Connected apps section
* Dynamic user profile support

##  UI Design

The project uses:

* Glassmorphism effects
* Neon gradients
* Smooth animations
* Responsive layouts
* Modern SaaS dashboard styling

##  Getting Started

Clone the repository:

```bash
git clone https://github.com/tanishkaadubey/Life-flow-AI.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
